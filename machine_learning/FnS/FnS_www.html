Goal:¶

To automatically flag warranty claims as possible customer safety concern using machine learning classification with Python's scikit-learn library

Current situation:¶

Some_Company_Division currently uses a large and complex query (implemented with IF/ELSE type of logic via SQL's CASE statement) to flag claims as customer safety issue. The study is to determine if we can apply machine learning classification to do the flagging instead. This study could potentially lead to other areas of warranty or market data that could benefit from machine learning classification.

The Data:¶

The data consists of 2 feature or attribute columns and a target or label column that would be used as input to train a machine learning algorithm.

Features columns:

  • warranty claim part # (text, categorical data type)
  • customer complaint description (free-hand text, document data type)

Target / label column:

  • A binary/2-value flag to indicate whether or not the claim is a potential safety concern

NOTE: This sample data used for training was purposely made to be small to help facilitate faster work flow. Inevitably, special methods will be needed if this process were to be applied in production with very large training data - see references at the very bottom of this notebook.

I discovered why I received this error. The sparse matrix for my test part number (X_new_part_onehot) has a shape of 1x1, where I needed it to be 1x153. So I need to set the n_values parameter to 153 (the number of unique numbers of part #s in my test data) when I one-hot encode the test part #. In production, I should determine the actual total number of unique part #s and use that to set the n_values parameter.

Corrected code:¶

In [30]:
part_test = np.array(['19301'])
complaint_test = np.array(['CLIENT STATES THE CHECK EMISSIONS LIGHT IS ON'])

X_new_part_labelencoded = enc_label.fit_transform(part_test)

# Need to instantiate a new OneHoteEncoder object and set the n_values parameter
# or elses we'll end up with a 1x1 matrix when we need to end up with a 1x153 matrix
enc_onehot_new = OneHotEncoder(n_values=153)
X_new_part_onehot = enc_onehot_new.fit_transform(X_new_part_labelencoded.reshape(-1,1))

X_new_complaint_counts = count_vect.transform(complaint_test)
X_new_complaint_tfidf = tfidf_transformer.transform(X_new_complaint_counts)

# Horizontally stack together the 2 sparse matrices
X_new_combined_tfidf = sparse.hstack((X_new_part_onehot, X_new_complaint_tfidf), format='csr')

predicted = clf.predict(X_new_combined_tfidf)
